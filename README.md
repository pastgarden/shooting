# シューティングバトルゲーム

2人で対戦して遊べる、  
シンプルなシューティング風Webゲームです。

## サイト

https://pastgarden.github.io/shooting/

## 内容

- 2人対戦用のシューティングゲーム
- 連打で弾を撃つ操作
- ブラウザ上で遊べるミニゲーム
- シンプルなバトル形式
- 操作性はあまり良くない仕様

## この作品について

HTML / CSS / JavaScript の練習として作った、  
2人対戦型のシューティングゲームです。

本格的なゲームというより、  
プレイヤー同士で弾を撃ち合って遊ぶ仕組みを  
Web上で作ってみることを目的にした作品です。

弾を撃つには連打が必要で、  
操作性はあまり良くありませんが、  
当時のゲーム制作練習として残しています。

## 使用技術

- HTML
- CSS
- JavaScript
- GitHub Pages

今見るとかなり粗い部分もありますが、
